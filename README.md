# 次元迷宮
![次元迷宮](https://garudoadventure.github.io/portfolioWeb/assets/title.png)

### 作品概要
2Dと3D、さらにレイヤーを切り替える能力を駆使して、迷宮に仕掛けられた謎を解き、<br>
脱出を目指すパズルアドベンチャーゲームです。

### 操作方法
| Input | Action |
| --- | --- |
| Right/Left | 移動 |
| Enter | 確定 |
| Z | ジャンプ |
| X | 会話 |
| Spcae | 2D/3D　切り替え |
| Space 長押し | レイヤー切り替え |
